句型

  1. No matter wh- %s, 主詞 動詞 %s
    讓步句：無論%s，都%s 說明：No matter 加上疑問詞（what, when, where, who, how 等）引導的子句置於句首，表示「無論%s」，主句為不受影響的結果。子句需使用正常陳述語序，不可倒裝。
    意思：「無論%s，都%s」、「不管%s，還是%s」
    例句：No matter where you are, you can watch tutorials for professional skills with the internet.
    無論你在哪裡，都可以透過網路觀看專業技能的教學影片。
  2. 主詞 keep/leave/find 受詞 Adj/V-ing/P.P.
    使役／狀態動詞 + 受詞 + 補語 說明：keep（使保持）、leave（使處於／遺留）、find（發現處於某狀態）之後接受詞，再接形容詞、現在分詞（主動進行）或過去分詞（被動完成）作為補語，描述受詞的狀態或動作。
    意思：
    • keep O C：讓 O 維持 C 狀態
    • leave O C：放任 O 處於 C 狀態
    • find O C：發現 O 處於 C 狀態
    例句：John leave Mancy playing the game.
    約翰放任曼西一直玩遊戲

句型

  1. Never be動詞 主詞 %s
    否定副詞置句首的倒裝（動詞為 be） 說明：當否定副詞（Never, Hardly, Rarely）置於句首時，主詞與 be 動詞交換位置，以強調否定。
    意思：「從來沒有…」、「很少…」（強調語氣）
    例句：Never is Enoa defeated by the ugliness of humanity.
    艾諾雅從未被世俗的醜陋所擊敗。
  2. Never 助動詞 主詞 原形動詞
    否定副詞置句首的倒裝（動詞為一般動詞） 說明：否定副詞置句首時，將助動詞移到主詞前，主動詞保持原形。若原句無助動詞，加入 do/does/did。
    意思：「絕不…」、「很少…」（強烈否定或稀有情況）
    例句：Never does Cindy stop learning every day.
    辛蒂從未停止學習。
  3. It is (high/about) time (that) 主詞 過去式動詞
    虛擬語氣：早該做某事了 說明：用過去式表達與現在事實相反，暗示「現在還沒做，但早該做了」。be 動詞一律用 were。
    意思：「早該…（卻還沒）」
    例句：It is high time that you went to defeat the boss.
    你早該去擊敗首領了。
